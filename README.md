# 保研夏令营信息追踪系统

一站式管理保研夏令营申请进度的工具，包含 **信息爬虫** + **Web 管理面板**。

## 功能

- **招生通知爬虫**：自动抓取 16 所高校/研究所的招生通知页面，筛选夏令营、预推免相关信息
- **Web 管理面板**：可视化展示申请进度、导师套磁状态、院校截止日期等
- **导师管理**：按学校-导师层级管理套磁邮件、回复记录
- **状态追踪**：自动检测申请状态（未开始 → 待套磁 → 已套磁 → 材料准备中 → 材料已提交 → 已入营）
- **定时监控**：macOS launchd 定时任务，每 12 小时自动抓取

## 覆盖院校

上海交通大学（AI学院、计算机学院）、北京大学（智能学院、计算机学院、软微学院）、清华大学（计算机系、电子系、自动化系、深圳国际研究生院）、浙江大学（计算机学院）、南京大学（智科院、计算机学院）、中科院（自动化所、计算所）、复旦大学（计算机学院）、中国人民大学（高瓴人工智能学院）...

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 初始化目录结构

```bash
# 创建院校文件夹（系统会自动检测）
mkdir -p 上交AI 上交计算机 北大智能 北大计算机 北大软微
mkdir -p 南大智科 南大计算机 浙大计算机 中科院自动化所 中科院计算所
mkdir -p 清华电子 清华自动化 清华计算机 清华深圳 复旦 人大高瓴

# 创建个人资料目录（放简历等）
mkdir -p 个人资料
```

### 3. 运行爬虫（首次）

```bash
python3 monitor/crawler.py
```

### 4. 启动 Web 面板

```bash
python3 webapp/app.py
# 访问 http://localhost:5208
```

### 5. （可选）设置定时任务

```bash
bash monitor/setup.sh
```

## 自定义配置

### 修改个人信息

编辑 `webapp/app.py` 中的 `api_profile()` 函数，填入你的个人信息。

### 添加/删除院校

在 `webapp/app.py` 中修改 `SCHOOL_FOLDERS` 和 `SCHOOL_INFO` 字典，同时在 `monitor/crawler.py` 的 `MONITOR_TARGETS` 中添加对应的监控 URL。

### 导师管理

在学校文件夹下创建以导师姓名命名的子文件夹，放入 `陶瓷邮件.md` 等文件，系统会自动识别导师并展示在 Web 面板中。

```
上交AI/
├── 张教授/
│   ├── 陶瓷邮件.md    # 套磁邮件草稿
│   └── 回复.md        # 导师回复记录
├── 李教授/
│   └── 陶瓷邮件.md
└── 夏令营通知.pdf      # 学院级资料
```
